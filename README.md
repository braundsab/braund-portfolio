# Stephen Braund Portfolio

Personal portfolio website showcasing communications strategy, editorial work, and photography.

Built with Jekyll and hosted on GitHub Pages.

## Site Structure

- **Homepage:** Featured work carousel and project highlights
- **Portfolio:** Filterable project grid with lightbox galleries  
- **About:** Background and capabilities
- **Blog:** Articles and insights
- **Contact:** Get in touch form

## Technologies

- **Static Site Generator:** Jekyll 4.4
- **Styling:** Custom SCSS with CSS variables for theming
- **JavaScript:** Vanilla JS (carousel, filtering, lightbox)
- **Hosting:** GitHub Pages
- **CDN:** Cloudflare
- **Forms:** Formspree

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000`
